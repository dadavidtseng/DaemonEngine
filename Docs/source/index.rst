Daemon Engine's official documentation
======================================================================================================
Daemon Engine is a developing game engine built with C++.

Features
------------------------------------------------------------------------------------------------------

* DirectX11 Render system
* Math library
* Multiple windows support
* Audio support with FMOD integration
