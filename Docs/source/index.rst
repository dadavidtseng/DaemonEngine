Welcome to Engine's documentation!
===================================

Engine is a developing 2D game engine built with C++.

Features
--------

* High-performance 2D rendering system
* Complete math library
* Modular core system architecture  
* Audio support with FMOD integration
* Cross-platform support

Getting Started
---------------

This documentation will help you understand and use the Engine framework.

Quick Start
-----------

To get started with Engine, follow these basic steps:

1. Clone the repository
2. Build the project using Visual Studio
3. Run the sample application

API Reference
-------------

The Engine provides several core modules:

* **Core System**: Basic functionality and utilities
* **Renderer**: 2D rendering and graphics
* **Math Library**: Vector operations and mathematical functions
